# NGS_SnakeMake

### SnakeMake workflow for variant calling

Tools include:
1. FastQC
2. Trimmomatic
3. BWA index
4. BWA-mem
5. SortSam
6. Picard MarkDuplicates
7. Samtools index, idxstats, flagstat, faidx
8. Freebayes
